Refactor duplicated Cluster Node data into a single shared inventory file.

Proposed shape:

```json
inventory/cluster.json
```

Store non-secret topology data only:

- network gateway/DNS/prefix
- Kubernetes API endpoint
- Cluster Node name -> IP
- VM ID
- role
- CPU/RAM/root disk
- Longhorn data disk size

Consumers:

- Terraform: `jsondecode(file(...))` for `local.cluster_nodes` and network settings.
- NixOS host configs: `builtins.fromJSON` to set host IP, gateway, DNS.
- `nix/modules/k3s-agent.nix`: read Kubernetes API endpoint instead of hardcoding `192.168.2.81`.
- Colmena hive/flake host inventory: derive target hosts from same inventory if practical.

Avoid storing secrets here.
